# Card Preview - Telegram Mini App

Прототип мини-приложения для Telegram на React.js для управления картами.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Структура

- `/` - Главная страница с картой, балансом, подписками и графиком
- `/history` - История транзакций с фильтрами и поиском
- `/topup` - Пополнение баланса через криптовалюты
- `/cards` - Управление картами
- `/settings` - Настройки безопасности, уведомлений и профиля

## Особенности

- Нижняя навигация на всех страницах
- Верхнее меню (кроме главной страницы)
- Адаптивный дизайн
- Интеграция с Telegram WebApp SDK
